# 连连看小游戏 · Lianliankan Game

一款基于 **原生 Canvas + JavaScript** 开发的经典连连看益智游戏，支持 **抖音小游戏** 平台发布，同时可在浏览器中直接运行预览。

---

## 🎮 游戏特色

- ✅ **5 个难度关卡**：从 6×6 到 10×10，逐步挑战
- ✅ **12 种彩色图案**：使用 emoji + 渐变色块，无需任何图片资源
- ✅ **完整的连连看核心算法**：直线 / 单拐点 / 双拐点路径检测
- ✅ **实时可解检测**：遇到死局自动重排
- ✅ **倒计时系统**：每关限时挑战，剩余时间转化为奖励分
- ✅ **提示 & 重排道具**：帮你卡住时破局
- ✅ **丰富的动画反馈**：选中发光、连线动画、抖动提示、脉冲缩放
- ✅ **零依赖**：纯 Canvas 2D 实现，无需任何第三方库
- ✅ **跨平台**：抖音小游戏 / 浏览器 双模式自动识别

## 🚀 快速开始

### 方式一：浏览器预览（推荐用于调试、快速体验）

无需任何构建工具，直接用浏览器打开 `index.html` 即可。

```bash
# 任选其一：
# 1. 双击 index.html
# 2. 或启动一个简易本地服务器（避免部分浏览器的本地文件限制）
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 方式二：抖音开发者工具（用于真机预览 & 发布）

1. 下载并安装 **「抖音开发者工具」**
2. 打开工具，选择「小游戏」→「导入项目」
3. 选择本项目根目录
4. 填入你的抖音小游戏 **AppID**（没有的话可选择「测试号」）
5. 点击「编译」即可在模拟器中运行
6. 点击「预览」扫码可在真机上测试

## 🎯 游戏玩法

1. 点击「开始游戏」进入第 1 关
2. **点击两个相同的图案**：
   - 若它们可以用「不超过 3 段的直线」连接（即最多 2 个拐点），则消除成功
   - 否则选中会切换到新点击的格子，并附带抖动提示
3. 消除棋盘上所有图案即为过关
4. 过关后会进入下一关（更大棋盘 + 更多图案种类）
5. 时间耗尽则游戏结束

**道具：**
- 💡 **提示**（3 次）：高亮显示一对可匹配的图案
- 🔀 **重排**（3 次）：将剩余图案随机重新排布
- 每过一关会获得额外道具奖励

**计分规则：**
- 每次消除：`10 × 当前关卡数` 分
- 过关奖励：`剩余秒数 × 5` 分

## 🧠 核心算法说明

### 1. 棋盘生成

- 保证每种图案成对出现（总数必为偶数）
- 使用 Fisher-Yates 洗牌算法随机分布
- 生成后会验证「至少存在一对可消除的」，否则重新生成

### 2. 路径检测（连连看核心）

依次尝试三种连通方式：

| 拐点数量 | 图示 | 检测逻辑 |
|---------|------|---------|
| **0 个** | 直线 | 同行或同列，中间无其它格子阻隔 |
| **1 个** | L 形 | 尝试 `(r1, c2)` 与 `(r2, c1)` 两个候选拐点 |
| **2 个** | Z/U 形 | 沿行列枚举，对每个候选行/列做双向直线检测 |

> 边界外的「虚拟空行」会被视为可通行区域，因此图案可以贴着棋盘边缘连接。

### 3. 可解检测

每次消除后，枚举所有剩余的同类型图案对，若存在任意一对可通过上述算法连通，则棋盘仍有解；否则自动重排。

## 🔧 自定义与扩展

### 修改难度（关卡配置）

在 `game.js` 中找到 `config.levels`：

```javascript
levels: [
  { rows: 6, cols: 6, types: 6, time: 120 },  // 第 1 关
  { rows: 6, cols: 8, types: 8, time: 150 },  // 第 2 关
  { rows: 8, cols: 8, types: 10, time: 180 }, // 第 3 关
  { rows: 8, cols: 10, types: 12, time: 200 },// 第 4 关
  { rows: 10, cols: 10, types: 12, time: 240 }// 第 5 关
]
```

### 修改图案种类

在 `game.js` 中找到 `ICON_TYPES`，替换 `emoji` 或颜色：

```javascript
const ICON_TYPES = [
  { color: '#FF6B6B', emoji: '★' },
  { color: '#4ECDC4', emoji: '●' },
  // ...
]
```

> **进阶**：若想换成图片素材，可将 `drawCell()` 函数中的文字绘制改为 `ctx.drawImage()` 调用，并在游戏启动时通过 `tt.createImage()`（抖音）或 `new Image()`（浏览器）预加载资源。

### 调整道具数量

修改 `startGame()` 中的：
```javascript
hintCount = 3;    // 初始提示次数
shuffleCount = 3; // 初始重排次数
```

## 📱 平台兼容性

| 平台 | 状态 | 入口/API |
|------|------|----------|
| 抖音小游戏 | ✅ 已适配 | `tt.createCanvas()`、`tt.onTouchStart()`、`tt.getSystemInfoSync()` |
| 微信小程序 | 🔧 可迁移（API 近似，主要改动：`tt.` → `wx.`） | 同左 |
| Chrome / Safari / 移动端浏览器 | ✅ 已适配 | HTML5 Canvas + Pointer/Touch 事件 |

> **迁移到微信小游戏**：只需全局替换 `tt.` 为 `wx.`，并更新 `game.json` / `project.config.json` 的字段即可。
